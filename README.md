## 项目详情：
1. 程序拥有登录，信息发布等功能
2. 一级目录包括，home（首页）、my（我的）
3. 拥有二级可视化界面，包括detail（详情页）、publish（发布页）
4. 拥有四个列表分类，包括全部，集市，树洞，其他
5. 拥有发帖、评论、回复评论功能
6. 发布分为实名，匿名两种方式
7. 发布通过四个标签对应四个分类，若为树洞则强制匿名，其余分类则实名、匿名自选
8. 要求统一风格，颜色，形状协调一直
